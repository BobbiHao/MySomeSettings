# keyboard bindings
## 1. 视图：切换集中终端 
{
  "key": "ctrl+`",
  "command": "workbench.action.terminal.toggleTerminal"
}
## 2. Open in Terminal 
{
  "key": "ctrl+alt+o",
  "command": "openInTerminal"
}
## 3. 终端：聚焦到上一终端
{
  "key": "alt+[",
  "command": "workbench.action.terminal.focusPrevious"
}
## 4. 终端：聚焦到下一终端
{
  "key": "alt+]",
  "command": "workbench.action.terminal.focusNext"
}
